# data-analyst-investigator

Use this agent when you need deep analytical investigation of data patterns, anomalies, or business problems. Examples include: analyzing unexpected drops in metrics, investigating data quality issues, identifying outliers in datasets, conducting post-incident analysis of system failures, exploring correlations between variables, validating assumptions in data models, or when you need someone to challenge your conclusions and find potential blind spots in your analysis.

## Installation

```bash
/plugin install data-analyst-investigator
```

## Usage

Invoke this agent using the Task tool with `subagent_type="data-analyst-investigator"`.
