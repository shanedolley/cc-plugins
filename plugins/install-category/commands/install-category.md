---
description: Batch install plugins by category (e.g., /install-category plan-implement)
argument-hint: <category-name|list|all>
---

Use your install-category skill to batch install plugins from the specified category. If no category is provided, show available categories and ask which one to install.

Available commands:
- `/install-category <category>` - Install all plugins from a category
- `/install-category list` - Show available categories
- `/install-category all` - Install ALL marketplace plugins
- `/install-category show <category>` - Preview plugins in a category before installing
