# reviewing-github-prs

Use when reviewing any GitHub pull request - teammate PRs, external contributions, cross-team reviews, or self-review. Provides comprehensive analysis with flexible output modes via gh CLI.

## Installation

```bash
/plugin install reviewing-github-prs
```

## Usage

This skill is automatically activated when relevant contexts are detected.
